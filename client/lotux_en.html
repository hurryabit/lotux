Lotux

Playing Lotus under Linux

Laws of the game

You can read the laws of the game under Help/How to play Lotus.

Game type

On the one hand you can play toghether with a friend on one Computer (Local match). On the other hand you can play with other people via network (Network game).

A mode which allows to play against the computer is still in development.

How to start a new game

First you have to choose a game type in the Options menu. Following you need to select New game in the Game menu. If you have chosen a Network match it may happen that you have to wait for an opponent.

How to move the pieces

If you have chosen a stack and want to move the piece on its top, you just need to click on this stack. But you cannot move a stone of your opponent.

If you want to move a piece from the start square there may appear a dialog that asks you where you want to insert it. If you decide for left it is put on the left part of the map (marked with a L) else it is put on the right part (R).

If you cannot survey the whole board because the stacks overlay you have the possibility to rotate it. This may happen via the mouse (just hold the left button pressed and move the mouse) or the arrow keys.

The local match

If you have decided in favour of a local match the owner of the black pieces may do the first move. After that the game goes as described in the laws of the game. (In the status bar is always a hint whose turn it is.)

Das Netzwerkspiel

After a network match was started the application tries to connect to the server. If their occur any errors it stops with an error message. If the connection to the server was established the match can almost start. You just have to wait for a second player.

Now the server exchanges the names of the players and tells them the colour of their pieces. The one with black pieces has to start the game. (In the status bar is alway a hint whose turn it is.)

Settings

Over the item Settings... in the Options menu you have the possibility to configure your program.

First, you can choose the language of your application. But changes are only applied after a restart.

Moreover, you have the chance to change the search path of the application. Normally it should point to the directory where the translation and help files are stored (in the subdirectories lang and help, respectively). If you have installed the program in the usual way you do not need to change anything. But if you installed it in an extraordinary way, for instance in your home directory, this may be very helpful.

In the tab window Identity you can tell the program your name and your e-mail adress. They are send to your opponent during each network game.

The tab window Server allows you to make various settings concerning the server for your network games. You can choose the host and the port, where 5180 is the default port.

Where can I play Lotus in the internet?

Usually there is running a server at lotux.homelinux.org from 6 am till 10 pm (CET). Furthermore there is a list with other servers on http://lotux.homelinux.org/.
